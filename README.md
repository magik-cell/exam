## 🚀 швидкий старт

Виконайте наступні команди у вашому терміналі:

### 1. Встановлення залежностей
Ця команда встановить TypeScript, `ts-node` та необхідні описи типів:
```bash
npm install

```

### 2. Запуск TypeScript проекту
Виконайте скрипт для запуску коду TypeScript:
```bash
npm run start

```
